# orange ink

Ежедневный коллаж из PNG-изображений, опубликованный на GitHub Pages.

Каждую ночь GitHub Actions запускает `generate.py`, который случайным образом раскладывает 4–7 картинок из папки `images/` на холсте 1080×1080 px с поворотом и тенью. Дата используется как сид — коллаж за конкретный день всегда воспроизводится одинаково.

## Как устроено

```
images/        ← исходные PNG (добавляй сюда свои картинки)
collages/      ← готовые коллажи YYYY-MM-DD.png (генерируются автоматически)
generate.py    ← скрипт генерации
index.html     ← веб-галерея с навигацией по дням
```

**generate.py** — принимает дату в формате `YYYY-MM-DD` (по умолчанию сегодня), выбирает случайную подборку изображений, накладывает их с произвольным масштабом, поворотом и мягкой тенью.

**GitHub Actions** (`.github/workflows/collage.yml`) — запускается по расписанию в 23:49 UTC, коммитит новый коллаж в `collages/` и пушит в репозиторий. Можно запустить вручную через `workflow_dispatch`, указав конкретную дату.

**Веб-галерея** — статическая страница на GitHub Pages. Показывает коллаж текущего дня, позволяет листать по дням и скачивать PNG.

## Запуск локально

```bash
pip install Pillow
python generate.py              # коллаж за сегодня
python generate.py 2025-01-15   # коллаж за конкретную дату
```

## Добавление изображений

Положи PNG-файлы в папку `images/` и запушь. Со следующей ночи они начнут появляться в коллажах.
